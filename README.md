# Global_SuperStore_Project

TASK DESCRIPTION:
A CSV dataset has been provided for analysis. Using Tableau,
the goal is to evaluate product sales worldwide by creating interactive dashboards and visualizations.
This includes identifying top-performing regions, sales distribution patterns, and potential areas for improvement or growth.

FOCUS:
To develop a geographic (map) view in Tableau for visualizing sales data based on location.

Key Steps / Methodology:

1. Initialize Workspace:
Open Tableau and create a new worksheet to begin the visualization process.

2. Plot Geographic Data:
Drag and drop Country and State fields from the Data pane to the Detail section of the Marks card to generate a geographic map view.

3. Apply Regional Focus:
Add Region to the Filters shelf and select South to focus the map on that specific area.

4. Visualize Sales Data:
Drag the Sales measure to the Color section on the Marks card to create a filled (choropleth) map representing sales intensity per state.
